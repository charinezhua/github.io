宽松还是紧缩，萧条来了吗？

发表于 2019-01-05 | 更新于: 2019-01-05 | | 浏览 次
字数统计: 1.9k | 阅读时长 ≈ 11

对2019年初降准以及对导致降准紧缩的主要原因：2008年次贷危机的传导的理解

对当前降准的理解

降准了，按理论可以把准备金降到零，但是，商业银行的坏账如果多（假如10%），那么准备金降低的话商业银行的风险就很大，也就是说随着降准，小的银行风险在增加。

举个例子：你存了100万到商业银行，商业银行只能放贷88万（准备金按12%缴纳给央行），现在你要取回100万给你亲戚还房贷，那么央行就要退回12%的准备金，你拿到100万，给你亲戚还了房贷，这100万还是还回了商业银行，没有流通出去贷款或融资再产生效益，对商业银行来说就是货币乘数变小了。

在债务危机中，市场紧缩，银根紧缩，烂账成堆，货币流速变慢，对商业银行来说，存钱的人少，取钱的人多，商业银行的资金就会缺乏。央行降准就是相对补充商业银行的流动资金。商业银行并没有拿这些钱去放贷，只是应付自身的合理流动性。你去银行贷款叫银行“放水”（要给央行缴纳准备金），你去银行取你的存款不叫放水（央行退还准备金）。只有经济好的时候是拿降准的钱去放贷，才是放水，才会在市场上不断流动。这几次降准都是紧缩，不是我们理解的放水，为的是保证商业银行宽松。这次降准后，未来几个月M2增加跌破8%是必然。

假如还会降准一次二次或三次，降一次准维持不了多久，窟窿有多大，谁也不知道，不要等无准可降了才防风险，那时来不及了。

对2008年次贷危机传导的理解

2008年美国次贷危机，美国应该进入大萧条，美联储采取了超级宽松政策，这个政策输出了危机。下面说怎么输出危机的：

1、产能过剩导致的通货紧缩

由于大量美元流入中国，并伴随着产能的流入，中国面对新技术和大量的美元，疯狂的扩充产能，导致产能过剩，引起通货紧缩，由于产能的过剩，产品价格下跌，导致更加疯狂的产量增加，用以弥补利润的下降，越生产，价格越低，企业越借债生产，整个中国陷入通缩中，美国输出美元停止后，中国产能达到顶点，全球商品价格下跌引爆了中国的通缩爆发。

2、过度负债

中国的产能扩充伴随着债务增加，过度的负债支撑着中国的产能扩充。

3、债务--通缩

当产能过剩引起的大通缩爆发，并伴随着过度负债，标准的债务-通缩发生，通缩遇上过度负债，是无解的世界难题。对于中国内部来说，企业负债需要低利率减轻痛苦，需要产品输出获取利润，这些都无法做到。全球化的今天，中国可以通过输出产品获取美元来输出通缩，表现在货币上就是我说的M2对接贸易顺差，汇率贬值加大出口，这时，美国在上海G20关闭了人民币贬值的幻想，要求贸易平衡停止了中国的产品输出，美国用了10年输出了危机，怎么再可能让中国把危机反送回美国，当美联储主席艾伦说“美国在我有生之年再也不会爆发危机”就是宣布了美国输出危机结束。

4、恶性通缩下的大萧条如何爆发的。

中国天量的债务需要货币宽松，美联储的紧缩断绝了这条生路，面对全球紧缩，中国无法降低利率，债务在膨胀，企业的产能过剩无法救助，一带一路是遥远的天上之水，只能用供给侧改革来化解，天量的M2无法输出（通缩无法输出），一次股灾，一次房灾，十三亿人用血肉之躯来抵挡这世纪之灾。当资产价格下跌引爆债务危机，就是大萧条的爆发，当资产价格下跌，银根紧缩（现在就在紧缩），资产被出售，债务被放大（因为利率无法降低），就会爆发债务--通缩螺旋式下跌，越卖资产，债务越增加，直到整个破产，中国大萧条发生。美国用一切手段杜绝了中国救助危机的可能，防止中国危机引向全球。

全面监管

全面监管，就是打掉影子银行，中国就四大行是政策银行，再的基本都是各种猫腻的挣钱，各种表外业务，能不受监管就不受监管，谁愿意在表内缴纳准备金？谁愿意烂账曝光？现在就是表外回表内，烂账曝光不说，各种资产都要按标准缴纳准备金，停贷不说，要么吸收存款，要么甩卖资产缩表，影子银行本来就是开发商的小金库，都监管透明了，开发商就很难搞到钱了，资管到2020年，就是说一些银行到2020年才能把屁股擦干净，大概就是只收不放，小银行大概基本就停贷了。

资管后就是规范金融机构合规，一般人就不参与什么乱炒金融了，理财慢慢减少，消失，金融机构干自己的合法合规买卖，基本慢慢回到以前了，从M2和社会融资就可以看出来，现在商业银行收的多贷的少，已经开始缩表了，早按资管新规运作了，要合规基本就是金融市场有个调整过程，具体反映就是各种停贷等，是商业银行的调整过程，影子银行慢慢被压缩。要说影响就是对开发商搞钱有影响，开发商搞不到钱就割肉甩卖，商业银行一般就压缩房贷这些了，因为监管后都要缴纳准备金，不可能把高利率吸收的存款便宜借给你。资管新规要跟货币政策分开看，具体利率，货币宽松或紧缩，这些是货币政策，资管新规主要对金融市场影响大，就是不让乱搞金融了，就这意思。
